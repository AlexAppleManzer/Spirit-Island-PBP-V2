# PBPV2

This repository contains a React + TypeScript frontend and an Express/tsx backend.

## Getting Started

### Frontend

1. Open a terminal and navigate to the frontend folder:

```bash
cd frontend
```

2. Install dependencies:

```bash
npm install
```

3. Start the frontend development server:

```bash
npm run dev
```

The frontend app starts on `http://localhost:5173` by default.

### Backend

1. Open a second terminal and navigate to the backend folder:

```bash
cd backend
```

2. Install dependencies:

```bash
npm install
```

3. Start the backend server:

```bash
npm start
```

The backend server runs from `backend/index.ts` using `tsx`.

### Notes

- Run frontend and backend in separate terminals.
- Start the backend first if the frontend depends on API or websocket services.
